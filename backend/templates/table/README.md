# 표 양식 템플릿

이 폴더에 HWPX 표 양식 파일을 넣으세요.

## 파일 목록

| 파일명 | 열 수 | 용도 |
|---|---|---|
| `table_3col.hwpx` | 3열 | 3개 항목 비교표, 일정표 등 |
| `table_4col.hwpx` | 4열 | 4개 항목 비교표, 점검표 등 |
| `table_5col.hwpx` | 5열 | 5개 항목 상세 분석표 등 |

## 각 양식 파일의 기대 구조

```
행 1: 헤더 행 (열 수만큼 셀)
행 2: 샘플 데이터 행 (실제 삽입 시 이 행이 교체 기준이 됨)
```

### 헤더 행

- 배경색, 폰트, 정렬 등 서식이 사전 설정된 상태로 제공
- 셀 내용은 플레이스홀더 텍스트 (예: `[헤더1]`, `[헤더2]`, ...) 또는 비어있어도 무방
- 코드에서 `TableData.headers` 값으로 덮어씀

### 샘플 데이터 행

- 데이터 행의 서식(폰트·크기·정렬·셀 여백)만 사용하고 내용은 교체됨
- 코드에서 `TableData.rows` 값으로 반복 생성함

### 셀 폭 (권장)

| 양식 | 전체 폭 | 각 열 폭 |
|---|---|---|
| table_3col | 142mm | 열마다 자유 설정 가능 |
| table_4col | 142mm | 열마다 자유 설정 가능 |
| table_5col | 142mm | 열마다 자유 설정 가능 |

> 전체 폭 142mm는 A4 기준 좌우 여백 30mm 제외 본문 폭과 일치합니다.

## 추가 주의사항

- 표 캡션(제목)은 표 위에 별도 문단으로 자동 삽입됩니다 (양식 파일에 포함하지 마세요).
- 열 수가 `TableData.column_count`와 일치하지 않으면 삽입이 거부됩니다.
- 현재 지원 열 수: **3, 4, 5열**. 다른 열 수 요청 시 오류 메시지 반환.
